Inventário do Ansible — Grupos e Sub-grupos
O inventário define quais hosts existem e como são organizados em grupos
/etc/ansible/hosts
# hosts individuais
192.168.1.100
# grupo web
[servidores_web]
192.168.1.101
192.168.1.102
# grupo banco
[servidores_bd]
192.168.1.200
# subgrupo (children)
[servidores:children]
servidores_web servidores_bd
[servidores:children]
