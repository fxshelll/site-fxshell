Ciclo de Vida do Container Docker
Estados possíveis e comandos de transição
📦
CREATED
Container criado mas não iniciado
🟢
RUNNING
Processo em execução em foreground
⏸️
PAUSED
Congelado via SIGSTOP no kernel
🔴
STOPPED
Processo encerrado, dados preservados
🗑️
REMOVED
Container e camada R/W apagados
comandos de ciclo de vida
docker container run -d nginx→cria e inicia
docker container start <id>→inicia parado
docker container stop <id>→para graciosamente
docker container restart <id>→reinicia
docker container pause <id>→congela processos
docker container unpause <id>→descongela
docker container rm <id>→remove parado
docker container rm -f <id>→remove forçado
